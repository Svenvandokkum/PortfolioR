Hive - Work in Progress

Hive is a board-game similar to chess, but more complex. As a competition between some friends, I am programming Hive in python and I also intend to make an AI that can play Hive. Currently the game is roughly 70% complete and I hope to finish soon!

Process

Making this game in python has two main difficulties: The game logic is quite complex and the game is plaxed on a hexagonal grid. To tackle this, I had to code through a smart object-oriented way. This involved things like designing a class for a coordinate system. The most learnful experience of coding this game was learning how to code cleanly, because without clean code it would be impossible to code something complex. After I am done with the game logic I will make the UI and graphics more pretty.
